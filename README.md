# fallbridge-sdk

**FallBridge · Web Bluetooth GATT terminal for BLE-mesh dongles** — sovereign SDK extracted from `fallbridge/index.html`.

## Install
```bash
npm install @ai-native-solutions/fallbridge-sdk
```

## Use
```js
import * as fallbridge from '@ai-native-solutions/fallbridge-sdk';
```

- Playground: https://sjgant80-hub.github.io/fallbridge-sdk/
- MCP wrapper: [`@ai-native-solutions/fallbridge-mcp`](https://github.com/sjgant80-hub/fallbridge-mcp)
- HTTP API: [`@ai-native-solutions/fallbridge-api`](https://github.com/sjgant80-hub/fallbridge-api)
